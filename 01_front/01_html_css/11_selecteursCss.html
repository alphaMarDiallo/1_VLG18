Propriété css utilisées dans cette page

Décrivez leur usage et leurs valeurs possible

  1. background : couleur du fond de l'élément
  2. color : couleur de la police
  3. float : effet flottant
  4. font-family : police
  5. font-size : taille de la police
  6. height : hauteur de l'élément
  7. left : position à partir de la gauche de la fenêtre
  8. line-height : hauteur de l'interligne
  9. margin : marges extérieures
  10. padding : marges intérieures
  11. position : position de l'élément dans la page (on n'a utilisé que la valeur (fixed) pour l'instant
  12. text-align : alignement du text
  13. text-décoration : changement des valeurs héritées (ici le souligné qui est standard pour les liens sous tous les navigateurs est supprimé
  14. top : position à partir du haut de la fenêtre
  15. transition : tempde transition lors d'une animation (ici sur les liens de la barre menus)
  16. width : largeurde l'élément par rapport à la taille de la fenêtre si on indique une valeur relative (%, em...)

Propriété css utilisées à rechercher

Rechercher les propriétés ci-dessous, faites des teste avec sur cette page depuis la console de débuggage pour tester les valeurs qu'elles peuvent prendre

  1. font-weight : permet de définir la graisse utilisée pour le texte.
  2. letter-spacing : définit l'interlettre utilisée pour les caractères qui composent le texte.
  3. word-spacing : définit la règle d'espacement utilisée entre les balises et les mots
  4. text-shadow : ajoute des ombres au texte. Chaque ombre est décrite par une certaine combinaison de décalages X et Y de l'élément, de rayon de flou et de couleur.
  5. background-image : permet de définir une ou plusieurs images comme arrière(s)-plan(s) pour un élément.
  6. background-size : définit la taille des images d'arrière-plan pour l'élément. La taille de l'image peut être contrainte, complètement ou partiellement afin de conserver ses proportions.
  7. opacity : permet de modifier la transparence de l'image. Le résultat obtenu par cette fonction est une valeur filter-function .
  8. border-style :permet de définir le style des lignes utilisées pour les bordures des quatre côtés de la boîte d'un élément.
  9. box-shadow :décrit les ombres appliquées à la boîte avec une liste d'ombres séparées par des virgules
  10. list-style-image : définit l'image utilisée comme puce devant les éléments de listes. On peut également utiliser la propriété raccourcie list-style.
  11. border : permet de définir les propriétés liées à la bordure. border peut être utilisée pour définir les valeurs de border-width, border-style, border-color
  12. border-radius :permet de définir des coins arrondis pour la bordure d'un élément. La courbure de chaque coin est définie avec un ou deux rayons de courbures qui permettent de définir un arc de cercle ou un arc d'ellipse
  13. border-width : définit la largeur d'une bordure pour la boîte d'un élément.

  15. background-color: permet de définir la couleur utilisée pour l'arrière-plan d'un élément (celle-ci peut être une couleur transparente).
